Espaços em Branco

Use espaços em branco para melhorar a legibilidade. Todo o código deve ser uniforme então nada de misturar tabulação com espaços.

Iremos manter o uso de 02 espaços para identação, tanto no markup (HTML, .hbs) quanto no CSS/SCSS.

Espaços em branco devem sempre ser usados antes de indicar o valor de uma propriedade CSS. Veja o exemplo abaixo:

/* Ruim */
.class {
height: 20px; /* Sem identação */
}

.class {
  height:20px; /* Sem espaço antes do valor da propriedade */
}

/* Bom */
.class {
  height: 20px;
}
        
<!-- Ruim: sem identação -->
<div class="whatever">
<p class="parent">
<!-- content -->
</p>
</div>

<!-- Bom -->
<div class="whatever">
  <p class="parent">
    <!-- content -->
  </p>
</div>
