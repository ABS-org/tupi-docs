Classes e Variáveis

Você não é um compilador/compressor de código humano, então não tente ser.

Use nomes claros e previdentes para classes HTML e variáveis SCSS. Escolha um padrão de nomes compreensível e consistente que faça sentido para arquivos HTML e arquivos CSS.

Além disso, alguns princípios devem ser seguidos. Veja abaixo:

  • Use hífen para nomes compostos e dois hífens para separar nomes do estado da classe, tipo etc.
  • Ao nomear classes, primeiro deve vir o tipo do elemento, seguido do nome propriamente dito.

Veja exemplos: 

[cc lang=”css”]
/* Ruim */
.component-awesome-blue {
  overflow: auto;
}

.awesome--component {
  overflow: auto;
}

/* Bom */
.component--awesome-blue {
  overflow: auto;
}

.component--awesome {
  overflow: auto;
}
