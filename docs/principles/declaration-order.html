Ordem de Declaração

Declarações devem ser ordenadas segundo um único princípio. Nesse caso, agruparemos as propriedades relacionadas, iniciando por propriedades referente a estrutura e posicionamento. Ex.: tamanho, espaçamento, margem etc. Em seguida, as propriedades referente ao estilo: cor, tamanho de fonte, fundo etc.

Caso seja incluído mixins ou includes nas declarações, estes devem vir antes, seguindo o mesmo princípio: estrutura antecedendo os estilos. 

/* Ruim */
.class {
  height: 100px;
  color: #fff;
  position: relative;
  background: #eee;
  width: 100px;
  top: 10px;
}

/* Bom */
.class {
  /* Estrutura */
  width: 200px;
  height: 100px;
  padding: 10px;
  margin: 0 auto;
  position: relative;
  top: 20px;
  /* Estilo */
  background: #eee;
  color: #fff;
}
