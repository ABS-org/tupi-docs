Exceções

Algumas exceções devem ocorrer em situações específicas, elas estão descritas abaixo:s

  • Grandes blocos de declarações individuais podem atuar de forma diferente, através da formatação de linha única. Nesse caso, um espaço deve ser considerado depois da abertura das chaves e antes do fechamento das chaves.
  • Usar valores de cor em hexadecimal, porém se houver necessidade de opacidade, usar RGBA.
  • Comentários em uma linha após a propriedade ou no meio da classe, são permitidos conforme o modelo
/* Comentários no meio da classe */
.class {
  height: 200px;
  overflow: auto; // Comentário básico de uma linha
}

/* Elementos com apenas uma propridade */
.class { width: 10%; }
.class { width: 20%; }
.class { width: 30%; }

/* Uso de RGBA, quando há necessidade de opacidade */
.class {
  background: rgba(255, 255, 255, 0.75); // Opacity only in background
}
