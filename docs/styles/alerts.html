Alerts

Os Alerts podem ser de vários estilos, indicando estados específicos. E podem ter transições ao fechar.

Exemplo

Alert Success

Alert Danger

Alert Warning

Alert Info (sem Fade in)

Aplicando os Estilos

Basta as classes alert-success, alert-danger, alert-warning ou alert-info para estilizar seu alert. É possível adicionar transições como fade, por exemplo.

Veja o código abaixo:

<div class="alert alert-success">
    Alert Success
</div>

<div class="alert alert-danger">
    Alert Danger
</div>

<div class="alert alert-warning">
    Alert Warning
</div>

<div class="alert alert-info fade in">
    Alert Info + Fade In
</div>

*Este componente é o mesmo do bootstrap, sem alterações significativas
