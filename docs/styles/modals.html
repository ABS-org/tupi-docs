Modals

As Modais podem ser de vários tamanhos e ter ou não transições.

Clique no botão para exibir a modal.

Aplicando os Estilos

Para indicar o tamanho de uma modal, basta adicionar as classes modal-sm ou modal-lg no modal-dialog.

Para usar o efeito fade, basta adicionar a classe fade onde estiver a classe modal.

Veja o código abaixo:

<!-- Modal com fade -->
<div id="modal-example" class="modal fade">
  <div class="modal-dialog modal-sm">
    <div class="modal-content">
      <p>Modal content<p>
    </div>
  </div>
</div>

<!-- Modal grande -->
<div id="modal-example" class="modal">
  <div class="modal-dialog modal-lg">
    <div class="modal-content">
      <p>Modal content<p>
    </div>
  </div>
</div>

<!-- Modal pequena -->
<div id="modal-example" class="modal">
  <div class="modal-dialog modal-sm">
    <div class="modal-content">
      <p>Modal content<p>
    </div>
  </div>
</div>

*Este componente é o mesmo do bootstrap, sem alterações significativas
