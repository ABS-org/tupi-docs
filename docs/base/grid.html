Grid

Abaixo veremos o modo de usar o grid do Tupi, heradado do Bootstrap

Grid semântico

As classes de grid não devem ser usadas inline, evitando a adoção de inúmeras classes, poluindo o markup. Nesse caso, devemos usar o grid de modo semântico, com apenas umas classe por elemento. Como nos exemplos abaixo.

/* Maneira correta de aplicar o grid */
.my-element {
  @include make-xs-column(12);
  @include make-sm-column(8);
  @include make-md-column(7);
  @include make-lg-column(6);
}
.element-wrap {
  @include make-row;
}
        
<!-- Maneira errada -->
<div class="col-xs-12 col-sm-8 col-md-7 col-lg-6">
</div>

<!-- Maneira correta -->
<div class="my-element">
</div>

<!-- Ou também -->
<div class="element-wrap">
  <div class="my-element">
  </div>
</div>

Perceba que é possível usar apenas uma classe e incluir os grids através do pré-processador CSS (SASS, utilizado pelo Tupi).

Caso seu elemento esteja inserido dentro de uma row (o que geralmente acontece), basta adicionar um elemento outro envolvendo. Este elemento deve receber o termo "wrap".
