Dropdowns

Dropdowns são menus suspensos que podem conter subitens, podem ser formulários pequenos ou outro tipo de conteúdo.

Exemplo

  • Link
  • Outro link
  • Link separado

Como usar

Basta criar um link (ou botão) com o atributo data-toggle contendo o valor dropdown, e um menu com classe dropdown-menu. Ambos envolvido com um elemento contendo a classe dropdown.

Veja o código abaixo:

<div class="dropdown">
  <a data-toggle="dropdown">
    Dropdown
  </a>
  <ul class="dropdown-menu">
    <li><a href="#item">Item</a></li>
  </ul>
</div>

*Este componente é o mesmo do bootstrap, sem alterações significativas
