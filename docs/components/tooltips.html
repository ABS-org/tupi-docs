Tooltips

Tooltips são uma versão melhorada do que é mostrado no atributo title padrão do HTML, usando animação em CSS3, JS e data-attributes para mostrar o title dos elemtos.

Exemplo

Como usar

Por razões de performance, as tooltips são uma funcionalidade opt-in, ou seja, você deve inicializa-las via javascript, ao contrário de outros componentes que são inicializados automaticamente.

Para inicializá-las você deve usar o atributo data-toggle="tooltip". Para complementar, você pode usar o atributo data-placement="top" para indicar onde a tooltip vai aparecer, que pode ter como valor top, left, right e bottom

As tooltips com o atributo title sem conteúdo não são mostradas. Ou seja, tooltips nunca vão aparecer vazias

Veja o exemplo abaixo:

<a data-toggle="tooltip" data-placement="top" title="Tooltip on top">
  Tooltip on top
</button>

<script type="text/javascript">
  $(function () {
    $('[data-toggle="tooltip"]').tooltip()
  })
</script>

*Este componente é o mesmo do bootstrap, sem alterações significativas
