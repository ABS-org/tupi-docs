Modals

Modais são uma espécie de janela suspensa na página. Geralmente são usada para confirmação ou exibir formulários de cadastro.

Clique no botão para exibir a modal.

Como usar

No link para a modal, deve ser incluído o atributo data-toggle com o valor modal e o atributo data-target com o id (ou classe) utilizado na modal, exemplo #modal-example.

No elemento que será transformado em modal, basta adicionar a class modal, mais um id qualquer, que deve ser o mesma do data-target.

Veja o código abaixo:

<!-- Link to Modal -->
<a data-toggle="modal" data-target="#modal-example">Link to Modal</a>

<!-- Modal -->
<div id="modal-example" class="modal">
  <div class="modal-dialog modal-sm">
    <div class="modal-content">
      <p>Modal content<p>
    </div>
  </div>
</div>

*Este componente é o mesmo do bootstrap, sem alterações significativas
