Tabs

Com as tabs é possível fazer transição entre paineis ou elementos em uma mesma página.

Exemplo

  • Tab 01
  • Tab 02
  • Tab 03

Como usar

As tabs são formadas por uma lista de links, onde um dos itens da lista deve receber a classe active e os links devem receber o href igual a seção referente a tab, e o atributo data-toggle recebendo o valor tab.

As seções devem estar envolvidas por uma tag cuja classe seja tab-content. Cada item deve conter um id, referente ao link das tabs, e a classe tab-pane. A classe active deve ser usada para indicar a seção que será exibida inicialmente.

Veja o código abaixo:

<!-- Tabs -->
<ul>
  <li class="active"><a href="#tab-02" data-toggle="tab">Tab 01</a></li>
  <li><a href="#tab-02" data-toggle="tab">Tab 02</a></li>
  <li><a href="#tab-03" data-toggle="tab">Tab 03</a></li>
</ul>

<!-- Tab panes -->
<div class="tab-content">
  <div class="tab-pane active" id="tab-01">Tab 01</div>
  <div class="tab-pane" id="tab-02">Tab 02</div>
  <div class="tab-pane" id="tab-03">Tab 03</div>
</div>

*Este componente é o mesmo do bootstrap, sem alterações significativas
